Version 2.0.0.0 (2007-07-17)
--
Family Data View with filtering, sorting, and in-place editing

Family Analytics including Last Name Tag Cloud, Age Distribution Historgram, 
  and Birthday List. Clicking on the items in each control filters the
  Family Data View.

Enhanced Story Editing with support for font name, size, alignment, bullets,
  and numerical lists
  
Filtering and sorting in the Family List view, in the main window

Skinnable user interface with two skins: black and silver

Improved tooltips in diagram view

Improved Windsor family sample data file with stories and images. Please note
  that the new Windsor family sample data file, stories, and images are only
  written out the first time Family.Show runs AND a Family.Show folder in
  the Documents folder does NOT already exist.
